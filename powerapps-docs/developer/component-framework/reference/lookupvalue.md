[!INCLUDE [lookupvalue-description](includes/lookupvalue-description.md)]

## Available for

Model-driven apps

## Properties

### id

The record ID.

**Type**: `string`

### name

The name of the record the lookup selection references.

**Type**: `string`

### entityType

The entity logical name of the record the lookup selection references.

**Type**: `string`
