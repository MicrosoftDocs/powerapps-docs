# FileObject

Provides access to all the properties of a file.

## Available for

Model-driven apps

## Properties

### fileContent

Contents of the file.

**Type**: `String`

### fileName

Name of the file.

**Type**: `String`

### fileSize

Size of the file in KB.

**Type**: `Number`

### mimeType

File MIME type.

**Type**: `String`
