# FileObject

[!INCLUDE [fileobject-description](includes/fileobject-description.md)]

## Available for

Model-driven and canvas apps.

## Properties

### fileContent

Contents of the file.

**Type**: `string`

### fileName

Name of the file.

**Type**: `string`

### fileSize

Size of the file in KB.

**Type**: `number`

### mimeType

File MIME type.

**Type**: `string`
