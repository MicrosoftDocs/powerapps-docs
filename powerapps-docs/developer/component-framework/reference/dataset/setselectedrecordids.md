## Syntax

`context.parameters.dataset.setSelectedRecordIds(ids)`

## Parameters

| Parameter Name | Type     | Required | description         |
| -------------- | -------- | -------- | ------------------- |
| ids            | `string` | Yes      | List of record id's |
