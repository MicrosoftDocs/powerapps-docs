# isOffline

[!INCLUDE [isoffline-description](includes/isoffline-description.md)]

## Syntax

`context.client.isOffline()`

## Available for

Model-driven and canvas apps

## Return Value

Type: `Boolean`
Description: **true** if the server is offline; **false** otherwise.
