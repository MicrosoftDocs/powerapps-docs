# getFormFactor

[!INCLUDE [getformfactor-description](includes/getformfactor-description.md)]

## Syntax

`context.client.getFormFactor()`

## Available for 

Model-driven apps and canvas apps (experimental preview)

## Return Value

Type: `Number`

|Value|Form Factor|
|---|---|
|0|Unknown|
|1|Desktop|
|2|Tablet|
|3|Phone|
