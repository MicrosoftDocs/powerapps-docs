## Syntax

`formatDecimal(value, precision)`

## Parameters

| Parameter Name|Type|Required|Description|
| ------------- |----|--------|-----------|
|value|`number`|yes|The date to be formatted.|
|precision|`number`|yes|The number of digits after decimal point.|

## Return Value

Type: `string`
