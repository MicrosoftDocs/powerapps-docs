# SocialProfile Entity Reference

This entity is used to store social profile information of its associated account and contacts on different social channels.

## Entity Properties

**DisplayName**: Social Profile<br />
**DisplayCollectionName**: Social Profiles<br />
**SchemaName**: SocialProfile<br />
**CollectionSchemaName**: SocialProfiles<br />
**LogicalName**: socialprofile<br />

**EntitySetName**: socialprofiles<br />
**PrimaryIdAttribute**: socialprofileid<br />
**PrimaryNameAttribute**: profilename<br />
**OwnershipType**: UserOwned<br />
**IsBPFEntity**: False<br />
