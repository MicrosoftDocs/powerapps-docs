# ReportEntity Entity Reference

Entities related to a report. A report can be related to multiple entities.

## Entity Properties

**DisplayName**: Report Related Entity<br />
**DisplayCollectionName**: Report Related Entities<br />
**SchemaName**: ReportEntity<br />
**CollectionSchemaName**: ReportEntities<br />
**LogicalName**: reportentity<br />
**LogicalCollectionName**: reportentities<br />
**EntitySetName**: reportentities<br />
**PrimaryIdAttribute**: reportentityid<br />
**PrimaryNameAttribute**: <br />
**OwnershipType**: None<br />
**IsBPFEntity**: False<br />
