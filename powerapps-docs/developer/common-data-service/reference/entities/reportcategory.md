# ReportCategory Entity Reference

Categories related to a report. A report can be related to multiple categories.

## Entity Properties

**DisplayName**: Report Related Category<br />
**DisplayCollectionName**: Report Related Categories<br />
**SchemaName**: ReportCategory<br />
**CollectionSchemaName**: ReportCategories<br />
**LogicalName**: reportcategory<br />
**LogicalCollectionName**: reportcategories<br />
**EntitySetName**: reportcategories<br />
**PrimaryIdAttribute**: reportcategoryid<br />
**PrimaryNameAttribute**: <br />
**OwnershipType**: None<br />
**IsBPFEntity**: False<br />
