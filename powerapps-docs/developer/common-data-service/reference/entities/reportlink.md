# ReportLink Entity Reference

Links and dependencies between reports. A report may drill through to another report, or it may have another report as a sub-report.

## Entity Properties

**DisplayName**: Report Link<br />
**DisplayCollectionName**: Report Links<br />
**SchemaName**: ReportLink<br />
**CollectionSchemaName**: ReportLinks<br />
**LogicalName**: reportlink<br />
**LogicalCollectionName**: reportlinks<br />
**EntitySetName**: reportlinks<br />
**PrimaryIdAttribute**: reportlinkid<br />
**PrimaryNameAttribute**: <br />
**OwnershipType**: None<br />
**IsBPFEntity**: False<br />
