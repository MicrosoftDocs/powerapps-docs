## Syntax

`updatePopup(name, newProps)`

## Parameters

| Parameter Name|Type|Required|Description|
| ------------- |----|--------|-----------|
|name|`string`|yes|The name of the popup you're trying to update.|
|newProps|`Popup`|no|The updated properties to give to the popup.|
