# getDefaultView (Client API reference)



Returns the ID value of the default lookup dialog view.

## Control types supported

Lookup

## Syntax
 
`formContext.getControl(arg).getDefaultView();`

## Return Value

**Type**: String

**Description**: ID of the default view.
