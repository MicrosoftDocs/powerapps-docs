# setLabel (Client API reference)



Sets the label of the control.

## Control types supported

All

## Syntax

`formContext.getControl(arg).setLabel(label);`

## Parameter

|Name|Type|Required|Description|
|--|--|--|--|
|label|String|Yes|The new label of the control.|
