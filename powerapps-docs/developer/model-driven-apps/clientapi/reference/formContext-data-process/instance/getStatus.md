[!INCLUDE[./includes/getStatus-description.md](./includes/getStatus-description.md)]

## Syntax

`formContext.data.process.getStatus();`

## Return Value

**Type**: String. 

**Description**:Returns one of the following values: **active**, **aborted**, or **finished**.
