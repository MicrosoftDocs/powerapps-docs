# isValid (Client API reference)



[!INCLUDE[./includes/isValid-description.md](./includes/isValid-description.md)]

## Syntax

`formContext.data.entity.isValid();`

## Return Type

**Type**: Boolean

**Description**: true if all of the entity data is valid; false otherwise.
