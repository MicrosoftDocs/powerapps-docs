# getIsDirty (Client API reference)



[!INCLUDE[./includes/getIsDirty-description.md](./includes/getIsDirty-description.md)]

## Syntax

`formContext.data.getIsDirty();`

## Return Type

**Type**: Boolean

**Description**: true if the form data has changed; false otherwise.
