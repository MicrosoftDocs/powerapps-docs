# data.isValid (Client API reference)



[!INCLUDE[./includes/isValid-description.md](./includes/isValid-description.md)]

## Syntax

`formContext.data.isValid();`

## Return Type

**Type**: Boolean

**Description**: true if all of the form data is valid; false otherwise.
