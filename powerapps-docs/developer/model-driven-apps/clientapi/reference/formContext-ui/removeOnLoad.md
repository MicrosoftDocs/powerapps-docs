## Syntax

`formContext.ui.removeOnLoad(myFunction)`

## Parameter

|Name|Type|Required|Description|
|--|--|--|--|
|myFunction|function reference|Yes|The function to be removed from the form [OnLoad](../events/form-onload.md) event.
